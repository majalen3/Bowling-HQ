# Data Needs Future

## Phase 3-4
- tracked ball path coordinates
- video-derived release metrics
- lane-to-lane transition differences
- carrydown observations after each game

## Phase 5+
- high-speed video for rev and axis extraction
- automated lane-map ingestion
- pin hit location and deflection measurements
- ball aging and resurfacing history
- model-training datasets with actual vs. predicted outcomes

## Why This Matters

These datasets let Bowling-HQ move from heuristic predictions to calibrated and eventually personalized physics models.
