# Section 9: Galaxy - Arsenal Visualization

## What is Galaxy?

**Galaxy** is your interactive 3D arsenal visualization system. It transforms your collection of bowling balls into a visual "galaxy" where each ball is a star, positioned by characteristics and performance data.

Think of it as your **equipment universe**—where you can:
- Visualize your entire arsenal
- See coverage relationships
- Identify gaps at a glance
- Compare balls in space
- Explore equipment synergies
- Plan strategic acquisitions

---

## How Galaxy Works

### The Galaxy Model

Each ball is plotted on multiple dimensions:

```
Primary Axes:
- X-axis: Oil Condition (Dry ←→ Heavy Oil)
- Y-axis: Hook Potential (Low ←→ High)
- Z-axis: Ball Speed Sensitivity (Forgiving ←→ Aggressive)

Size/Color represents:
- Size: RG (Radius of Gyration) value
- Color: Cover stock type
- Brightness: Your personal average with ball
- Animation: Usage frequency
```

### Visual Layout

```
                 HIGH HOOK
                    ↑
                    │
        Pearl ●     │     ● Hammer Ninja
       Reactive │   │      │ (Reactive)
                │   │      │
    Heavy Oil ← │───┼───→ Dry Lanes
    (Left)      │   │      (Right)
                │   │      
              ● │   │ ●
          IQ Tour│  │ Phaze II
            (Reactive)│
                │   │
                ↓   ↓
              LOW HOOK
              (Plastic)

Size = RG Value (larger = higher RG)
Color = Cover stock
Brightness = Your average
```
