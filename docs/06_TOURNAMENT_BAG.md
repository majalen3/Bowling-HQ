# Section 6: Tournament Bag - Strategic Lineup Builder

## What is Tournament Bag?

**Tournament Bag** is your strategic equipment planning tool. It helps you build the optimal combination of bowling balls for any competition—maximizing coverage, minimizing gaps, and ensuring you're prepared for every possible condition.

Think of it as your **tactical equipment strategist**—helping you:
- Identify coverage gaps
- Build strategic lineups
- Plan contingencies
- Optimize equipment choices
- Prepare for tournaments


## Tournament Bag Strategies

### The Conservative Bag
**Goal:** Maximize strike percentage, minimize risk

```
Ball 1: Most consistent ball (your ace)
Ball 2: Spare ball (plastic)
Ball 3: Backup option

Strategy: Trust one ball, keep score high
Best for: Tournament where strikes matter most
```

### The Versatile Bag
**Goal:** Handle any condition with adaptability

```
Ball 1: Balanced all-rounder
Ball 2: Dry lane specialist
Ball 3: Oil specialist
Ball 4: Spare ball

Strategy: Switch based on conditions
Best for: Unknown conditions, long tournaments
```

### The Aggressive Bag
**Goal:** Strike as much as possible, risk open frames

```
Ball 1: Strong hook specialist
Ball 2: Aggressive backend ball
Ball 3: Speed control ball
Ball 4: Spare ball

Strategy: Attack the pocket, take risks
Best for: When you need strikes to win
```

### The Defensive Bag
**Goal:** Minimize scoring variations, consistent finishes

```
Ball 1: Reliable workhorse
Ball 2: Spare ball (plastic)
Ball 3: Backup workhorse
Ball 4: Spare ball (plastic)

Strategy: Play safe, consistent
Best for: Match play where consistency wins
```

## Bag Evolution

### Beginner Bag (One Strike Ball)
```
1 strike ball (versatile)
1 spare ball (plastic)
Limited options, good for learning
```

### Intermediate Bag (Two Strike Balls)
```
2 strike balls (different characteristics)
1 spare ball
More flexibility, better for varied conditions
```

### Advanced Bag (Three Strike Balls)
```
3 strike balls (dry, medium, oil specialists)
1-2 spare balls
Professional-level coverage
```

### Expert Bag (Four+ Strike Balls)
```
4+ strike balls (specialized coverage)
Multiple spare balls (different finishes)
Complete coverage for any condition
```
