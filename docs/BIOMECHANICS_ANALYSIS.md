# Biomechanics Analysis

## Release Variables

- **Speed** changes skid and entry angle.
- **Rev rate** changes how quickly the ball transitions.
- **Axis rotation** changes backend shape.
- **Axis tilt** affects roll timing and continuation.
- **Consistency** determines uncertainty more than ceiling.

## Practical Bowling-HQ Use

Bowler metrics should feed every recommendation because the same pattern and ball can produce very different motion for:
- speed dominant bowlers
- matched bowlers
- rev dominant bowlers

## Minimum Data Capture

Immediate capture targets:
- ball speed
- rev rate estimate
- axis rotation estimate
- starting board / breakpoint board
- whether the shot was flush, light, or high

Future expansion targets:
- frame-by-frame release variance
- drift and launch-angle tracking from video
- fatigue indicators across blocks
