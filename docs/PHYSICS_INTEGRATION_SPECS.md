# Physics Integration Specs

## Commander AI Physics Engine Spec
- Input: pattern, environment, bowler release state, available arsenal
- Output: ranked ball fits, launch window, adjustment advice, confidence

## Ghost Bowler Baseline Spec
- Input: historical average, release profile, condition tags
- Output: expected score band and deviation profile per condition family

## Digital Twin Simulation Spec
- Input: scenario definition, chosen ball, bowler profile, lane evolution settings
- Output: Monte Carlo score range, strike rate, recommended contingencies

## Pattern Intelligence Core Spec
- Input: pattern metadata plus transition observations
- Output: difficulty score, breakpoint forecast, transition rate

## Maxwell AI Venue Model Spec
- Input: center surface data, machine notes, seasonal trends, lane history
- Output: venue-adjusted friction assumptions and reliability weighting
