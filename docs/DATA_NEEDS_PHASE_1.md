# Data Needs Phase 1

Collect these first because they unlock the physics engine with the least implementation risk.

## Required Reference Data
- Oil pattern catalog with length, volume, ratios, source, and validity window
- Ball catalog with coverstock, RG, differential, mass bias, grit, and status (`active`, `discontinued`)
- Lane surface metadata per center

## Required Session Data
- Bowler average
- ball used
- speed estimate
- rev rate estimate
- starting board
- breakpoint board
- result classification (flush / light / high / miss)

## Immediate Outputs Enabled
- pattern difficulty score
- best-fit ball ranking
- expected score range
- transition alerting
